Add an option to your WooCommerce products to enable helium adding. Optionally charge a fee. For WooCommerce 2.2+

== Description ==

This simple extension allows you to add a helium addon option to your products with optional price.

WooCommerce > Settings lets you set up the default helium-addable status per-product and cost. These costs can be overridden per-product too.

If the customer adds helium addon, it is shown in cart + orders, and adds your defined cost to the item cost too.

The plugin is compatible with Aelia Currency Switcher and WooCommerce Currency Converter from WooThemes.

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it :)
